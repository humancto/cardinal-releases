# Cardinal — Releases

Public releases of **Cardinal** — an interactive handbook of early math, designed for autistic children and the families who learn alongside them.

## Download

→ **[Cardinal-latest.dmg](Cardinal-latest.dmg)** — the latest macOS build.

Older releases are kept by version, e.g. `Cardinal-20260504.dmg`.

## Install

1. Open `Cardinal-latest.dmg`.
2. Drag `CardinalDemo.app` onto the `Applications` shortcut.
3. Open Applications, right-click on Cardinal, choose **Open**, click **Open** in the dialog.
4. Cardinal is now installed. Regular double-click works from now on.

The first-launch right-click is required because Cardinal is currently signed locally rather than notarised by Apple. We will move to notarised builds before the public launch.

## Privacy

Cardinal needs **no signup, no account, and never connects to the internet at runtime**. Your child's progress stays on your Mac. There is no analytics SDK, no telemetry, and no third-party tracking.
