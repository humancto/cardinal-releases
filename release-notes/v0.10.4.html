CARDINAL · v0.10.4

Heart icon owns the corner

A small visual fix: the favorite-heart icon and the chapter-card status pip ("READY" / "OPENED") were competing for the same corner. Removed the pip — the card's border opacity already conveys whether a chapter has been opened.

What changed

  • Chapter cards no longer show a "READY" / "OPENED" text label. Opened chapters have a more-saturated accent border; ready chapters have a softer border; coming-soon chapters have a hairline grey border. Visual hierarchy intact, less chrome.
  • Heart icon now has the upper-right corner unambiguously to itself — no more geometry-vs-text negotiation depending on column width.
  • The list view (compact rows) keeps the pip in flow — only grid cards lose it.

What didn't change

  • All v0.10.x features: Practice tab full coverage, Region picker, calm-landing onboarding, Sparkle auto-update, universal binary.
  • 217 self-checks passing.
  • Notarized + Developer ID signed + EdDSA-signed for Sparkle.
